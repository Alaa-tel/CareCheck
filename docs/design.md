# Design Approach

## Design System

This project uses Material Design principles as a foundation, adapted for a simplified healthcare experience.

---

## Key Decisions

### Simplified Material Usage
- Used basic patterns: buttons, cards, chips
- Avoided complex components (menus, drawers)

---

## Color

- Primary: Blue (trust)
- Background: White / light gray
- Status colors:
  - Low: Green
  - Medium: Yellow
  - High: Red

---

## Components

- Full-width buttons
- Symptom chips
- Cards for results

---

## Interaction Patterns

- One question per screen
- Tap-based interaction (no typing)
- Clear primary actions

---

## Mobile Considerations

- Designed for one-handed use
- Large tap targets
- Minimal content per screen
