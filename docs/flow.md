# User Flow

## Step 1: Home
- CTA: Start check

## Step 2: Symptom Selection
- User selects symptoms
- CTA: Continue

## Step 3: Question 1
- Example: Do you have a fever?

## Step 4: Question 2
- Example: How long have you had symptoms?

## Step 5: Summary
- Display selected symptoms + answers

## Step 6: Results
- Show recommendation:
  - Low → Self-care
  - Medium → Schedule
  - High → Urgent care
