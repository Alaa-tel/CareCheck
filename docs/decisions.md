# Design Decisions

## Symptom Selection

Used chip-based selection instead of text input:
- Faster on mobile
- Reduces typing

---

## Question Flow

One question per screen:
- Reduces cognitive load
- Easier to focus

---

## Simplified Logic

Triage logic is intentionally simple:
- Focus on UX clarity
- Not medical accuracy

---

## Result Screen

Provides:
- Clear recommendation
- Actionable next step

---

## Tone

Language is:
- Calm
- Clear
- Non-alarmist

---

## Navigation

Simple linear flow:
- Avoids confusion
- Keeps user moving forward
